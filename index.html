Eden for Eve Online

Eden is a character monitoring application for Android devices. You may download the current release from the play store, or build it yourself via source.

Libraries

Eden makes use of the following libraries:

  • SlidingMenu
  • Picasso
  • ActionBarSherlock
  • eveapi
